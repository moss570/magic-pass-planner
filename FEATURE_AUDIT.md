### Tier Positioning

**Pre-Trip Planner:** Entry-level, trip prep only
- Missing: Live alerts, AP tools, social, games
- Good for: First-time visitors, budget-conscious
- Weakness: Limited differentiation from free tools

**Magic Pass:** Full-featured platform
- Has: Everything except AP-specific tools
- Good for: Regular visitors, power users
- Strength: Most features, best value

**AP Command Center:** AP basic tier
- Has: AP-specific tools, discount database
- Missing: Live alerts, social, games, routing
- Weakness: Expensive for limited features ($7.99 vs Pre-Trip $6.99)

**AP Command Center PLUS:** AP premium tier
- Has: All AP tools + live alerts + routing + games
- Most complete for Annual Passholders
- Strength: Best value for frequent visitors
